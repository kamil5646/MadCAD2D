MadCAD 2D

Prosta aplikacja CAD 2D na Windows i macOS do projektowania konstrukcji stalowych: bramy, ogrodzenia, balkony.

Pobierz Windows (.exe) Pobierz macOS (.zip) Release 1.0

Rysowanie 2D

Linia, polilinia, prostokąt, okrąg, zaznaczanie, kopiowanie, przesuwanie i offset.

Wymiarowanie

Wymiary wyrównane, liniowe, obrócone i kątowe z konfiguracją stylu.

Generator stali

Szablony: brama, ogrodzenie, balkon. Automatyczne rozstawy paneli i profile.
