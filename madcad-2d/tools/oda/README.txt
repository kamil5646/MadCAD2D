MadCAD 2D - Dodatek ODA File Converter

Ten folder służy do bundlowania konwertera DWG/DXF razem z aplikacją.

Co wstawić:
- macOS: plik wykonywalny ODAFileConverter
- Windows: plik ODAFileConverter.exe

Jak to działa:
- Podczas startu i importu/eksportu DWG aplikacja najpierw szuka konwertera tutaj:
  1) resources/tools/oda (w zbudowanej aplikacji)
  2) tools/oda (w trybie developerskim)
- Jeśli plik istnieje, DWG działa bez dodatkowej konfiguracji.

Uwaga:
- Ten plik README możesz zostawić. Nie przeszkadza w działaniu.
